L'Infini + 1 = L'Infini

Le paradoxe de l'Hôtel de Hilbert, imaginé par David Hilbert en 1924, illustre la nature contre-intuitive des ensembles infinis.

1. Le Problème

Imaginez un hôtel avec une infinité de chambres (1, 2, 3...).
L'hôtel est complet. Chaque chambre est occupée. Un nouveau client arrive à la réception. Le gérant peut-il lui trouver une chambre sans mettre personne dehors ?

2. La Solution Simple (n → n+1)

Oui. Puisque l'infini n'est pas un nombre fini, on peut "pousser" tout le monde. Le gérant demande à chaque client de se déplacer de sa chambre n vers la chambre n + 1.

Chambre 1 → Chambre 2
Chambre 2 → Chambre 3
Chambre 3 → Chambre 4
...
Chambre n → Chambre n + 1

Résultat : La chambre 1 devient vide pour le nouvel arrivant. Personne n'a été expulsé, pourtant nous avons logé une personne de plus.

3. Le Bus Infini (n → 2n)

Maintenant, imaginez qu'un bus contenant une infinité de passagers arrive. Comment loger une infinité de personnes dans un hôtel déjà complet ?

Le gérant demande aux clients actuels de se déplacer dans la chambre correspondant au double de leur numéro actuel.

Chambre 1 → Chambre 2
Chambre 2 → Chambre 4
Chambre 3 → Chambre 6
...
Chambre n → Chambre 2n

Résultat : Toutes les chambres paires (2, 4, 6...) sont occupées par les anciens clients. Toutes les chambres impaires (1, 3, 5...) deviennent libres. Comme il y a une infinité de nombres impairs, tous les passagers du bus peuvent s'y installer !
